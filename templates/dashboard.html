More about credit sentinel

This project is designed to detect fraudulent credit card transactions using advanced machine learning models. It provides a secure platform for users to upload transaction data and receive fraud analysis results.

Project Overview

The application combines web development with Flask as a backend, HTML/CSS for frontend, and machine learning models for credit card fraud detection.

Technology Stack

  • Backend: Flask
  • Frontend: HTML, CSS (Bootstrap and Tailwind CSS), JavaScript
  • Machine Learning: Scikit-learn models (Isolation Forest, SVM, Logistic Regression)

Key Features

  • CSV file upload for transaction data
  • Fraud detection using machine learning models
  • Responsive design with Bootstrap and Tailwind CSS
  • Smart FAQ: Interact with the model to receive answers to your questions about credit card fraud detection and the application.
